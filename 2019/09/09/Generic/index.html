理解泛型

发表于 2019-09-09 | 分类于 notebook

序言

泛型在程序编写的过程中尤为重要，它不仅可以提升系统的可扩展性能，同时对于开发效率也有很大的帮助。

泛型介绍

从字面上理解，泛型是泛指的类型，在编程的过程中，不明确具体的数据类型，而是将类型作为参数的形式，传递给需要用到的对象。泛型使用主要包括以下几个方面，也将从这几方面去介绍泛型的使用场景：

  • 泛型类
  • 泛型方法
  • 泛型接口
  • 泛型擦除
  • 泛型通配符
