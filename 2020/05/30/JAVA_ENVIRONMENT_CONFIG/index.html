编程环境基础配置

发表于 2020-05-30 | 分类于 notebook

序言

本章记录新装系统搭建编程环境过程，编程语言以 JAVA为主，包括JDK，Git，Maven等常用环境及工具的配置

一、JDK 安装及环境部署

1.JDK 下载

这个直接去官网下载即可。

2.安装（JDK 1.8）

JDK 的安装一般默认即可，如果需要自定义，可以指定安装路径。

3.环境配置

在电脑属性的系统高级变量配置下，选择环境变量配置，新增及编辑如下变量。

JDK 环境配置，主要配置以下三项：

  • JAVA_HOME
    需新增，JDK 的安装路径：如：C:\Program Files\Java\jdk1.8.0_171
  • CLASSPATH
    需新增，JRE 工具包路径：如：%JAVA_HOME%\bin;%JAVA_HOME%\lib
  • Path
    已存在，配置服务及软件的bin启动支持：后面新增变量，如：%JAVA_HOME%\bin;C:\Program Files\Java\jre1.8.0_171\bin

验证安装结果

cmd 打开提示符命令框，输入java -version查看安装的jdk版本信息

分别输入java，javac，查看输出结果，正确安装会输出java相关的命令操作指令集

二、Maven 安装及环境部署

1.下载

这个直接去官网下载即可。

需要注意下载的压缩包有bin和src两种格式，bin是我们需要的，src是Maven的源码包。

2.安装（解压）及配置

解压到指定目录下即可。

在JDK的同样的系统环境变量配置中，配置如下：

  • MAVEN_HOME
    新增MAVEN_HOME变量，配置解压路径，如：E:\Program Files\apache-maven-3.6.3
  • Path
    在Path后新增Maven的命令启动支持，如：%MAVEN_HOME%\bin
